# Sweetp Dashboard

Chrome app for sweetp.

## Install

*   `npm install`
*   `bower install`
*   `grunt compass`
*   open chrome
*   go to extension managment
*   enable debug mode
*   click on "load unpacked extension"
*   select the `app` folder

## Dev

*   make sure SASS files ar compiled into css with
    *   `grunt compass` or
    *   `grunt server` or
    *   `grunt watch`
